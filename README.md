# Sproutime!

Sproutime! is a web-based gamified focus tool designed to help students and learners reduce procrastination by turning planned tasks into virtual planting activities.

## Project Summary

Users can create tasks, add estimated durations and subtasks, optionally use AI-supported subtask suggestions, receive a random seed, plant it in a 20-slot virtual garden, complete a focus session, water the plant by completing subtasks, and receive either a flower for full completion or a bud for partial completion.

## Main Features

- User registration and login
- Task creation with estimated duration and subtasks
- Optional AI-supported subtask suggestions
- Random seed assignment
- 20-slot virtual garden
- One active planted task at a time
- Focus timer with pause and resume
- Water Plant action for completed subtasks
- Flower or bud result logic
- Dashboard and task history
